ABOUT


          This is a very simple deep learning application which can detect up to 6 types of diseases 
          after studying sample medical images. The diseases included in this application are — Malaria
          , Brain Tumor, Optical Coherence Tomography, Diabetes Retinopathy, Breast Cancer and 
          Pneumonia using chest x-ray images.

          This is a beta version developed under the MIT open source license. Please feel free to 
          re-use and re-distribute the code. Feel free to give feedback and suggestions. Your feedback 
          will help us learn more and improve our application

          Of all the models, the malaria detection has the largest accuracy of 97%, the brain tumor has 
          the least accuracy of 65%. This is largely due to data constraints. I had very limited data 
          samples for my brain tumor models. Will try to improve upon the models in future updates. 
          The application is not fully optimized to run at the fastest speed and handle multiple client 
          requests using socket programming. If anyone has an idea on these topics, feel free to use them. 
          I will try it in the future updates. 

                                                    FUTURE WORK:
          I have plans to train several other models on medical datasets and add more useful features to 
          this app. For example, I tried implementing the google maps api for nearest hospital suggestions 
          this time but it hasn’t been fully implemented. I will keep them for future updates. I have planned 
          to build a separate app for deploying the YOLO model using openCV, so that you can use it for a 
          wide range of things — from having fun to having video surveillance. In this application, we will 
          try to improve the accuracy of the base models further. We have planned to write this in angular in 
          future so that it becomes much more flexible and you get a fast and cleaner UI. Meanwhile, if anyone 
          is interested to team up please drop us us an email. Any ideas or suggestions are welcomed.
